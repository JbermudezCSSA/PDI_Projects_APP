# Aplicación de Registro de Horas (PDI)

Esta es una aplicación web basada en Python y Streamlit diseñada para sustituir el antiguo sistema de control de horas en Excel. 

Permite a los usuarios registrar las horas dedicadas a cada proyecto y controla automáticamente que no se superen los límites diarios de jornada.

## Tecnologías empleadas:
* **Frontend/Backend:** Python + Streamlit
* **Gestión de datos:** Pandas
* **Base de datos:** Supabase (PostgreSQL)
